Some Important Question & Answers.

1. What is DOM and what is the purposes of DOM?

The Document Object Model (DOM) is a process in which the full html file is loaded as a Model of objects so that all the elements can be easily accessible. Using Javascript the DOM object can be modified dynamically.

There are many purposes of DOM. Such as:-

  1. HTML elements in the page can be changed dynamically using JavaScript.
  2. HTML attributes can be changed dynamically using JavaScript.
  3. CSS style can also be changed using DOM property.
  4. Can remove or add new HTML elements and attributes using DOM property.
  5. Various listener and actions can be added on various elements using selecting DOM elements using JavaScript.
2. How will you select HTML elements using DOM? Name the DOM methods.

There are 5 ways to select HTML elements using DOM property. They are :-

  • getElementsByTagName()
  • getElementsByClassName()
  • getElementByID()
  • querySelector()
  • querySelectorAll()
3. What is event bubble?

Event Bubble is a concept of DOM where any element receives any event and it bubbles up or propagate until it finds its parent or ancestor. Using stopPropagation() method, we can stop event bubbling. To find out the root of any element, we can use event bubble concept.

4. What is a callback function and why will you use it?

A callback function is a function which will be executed after executing another function. On other words, Any function that is passed as an argument to another function so that it can be executed in that other function is called as a callback function.

const button = document.getElementById('button');
function callback(){
console.log("Hello world");
}
button.addEventListener('click',callback);

Here, eventListener() function passed another function callback as an argument. So, callback() is the callback function here.
